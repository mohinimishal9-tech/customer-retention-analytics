# 🏦 Customer Engagement & Retention Analytics Dashboard

## 📌 Project Overview
An interactive analytics dashboard built using Python and Streamlit
to analyze customer churn behavior using the European Central Bank dataset.

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas
- NumPy
- Plotly

## 📊 Dashboard Features
- Engagement Profile Classification
- 5 KPI Metrics
- Churn Rate by Engagement Profile
- Product Utilization Impact Chart
- High-Value Disengaged Customer Detector
- Relationship Strength Index (RSI) Distribution

## 🚀 How to Run
```bash
pip install streamlit pandas numpy plotly
streamlit run app.py
```
